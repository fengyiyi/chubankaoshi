# 出版专业技术人员职业资格（中级）考试模拟系统

## 项目简介

本系统是针对出版专业中级职业资格自考的线上模拟考试网站，基于原生 PHP + SQLite 开发，完全本地运行，无需联网。

## 技术栈

- **后端**：PHP 7.4+ / PHP 8.0+（原生开发，无框架）
- **数据库**：SQLite（单文件数据库，免配置）
- **前端**：HTML + CSS + JavaScript + AJAX
- **运行环境**：支持本地服务器、虚拟主机，需开启 SQLite 扩展

## 功能特性

### 账号登录模块
- 默认固定账号：`admin`，密码：`123456`
- 30 天免登录功能（Cookie 自动记忆）
- 未登录用户强制跳转登录页
- 手动退出登录清除所有会话

### 核心考试功能
- **智能组卷**：按官方标准自动组卷（总分 200 分）
  - 单项选择题：30 题 × 1 分 = 30 分
  - 多项选择题：20 题 × 2 分 = 40 分
  - 综合应用题：5 题 × 26 分 = 130 分
- **断点续考**：实时保存答题进度，支持异常中断后继续
- **防重复组卷**：存在未完成考试时锁定新试卷生成
- **提交校验**：检测未作答题目并提示确认
- **自动评分**：提交后批量阅卷，统计得分和正确率

### 考试界面功能
- 75 分钟倒计时（最后 5 分钟闪烁提醒）
- 答题卡导航（已答/未答/标记状态）
- 题目标记功能
- 任意切换题目
- 随时修改答案

### 历史记录模块
- 永久存储所有考试记录
- 查看历次考试成绩（总分、得分、正确率）
- 支持回看试卷详情和答案解析

## 安装部署

### 环境要求
- PHP 7.4 或更高版本
- PHP SQLite 扩展已启用
- 写入权限（用于创建数据库文件）

### 快速启动

#### 方法一：PHP 内置服务器
```bash
cd /workspace
php -S localhost:8080
```
然后访问 http://localhost:8080

#### 方法二：XAMPP/WAMP
1. 将项目文件复制到 htdocs 或 www 目录
2. 启动 Apache 服务
3. 浏览器访问 http://localhost/项目目录

### 首次运行
系统会自动创建 SQLite 数据库文件 `exam.db` 并初始化示例题库。

## 使用说明

### 登录系统
1. 访问网站首页，自动跳转登录页
2. 输入账号 `admin` 和密码 `123456`
3. 勾选"30 天内免登录"（默认勾选）
4. 点击登录进入系统

### 开始考试
1. 在主页点击"开始新考试"
2. 系统自动按官方标准组卷
3. 进入考试界面开始答题

### 答题操作
- 点击选项选择答案（单选直接选，多选可多选）
- 使用"上一题"/"下一题"切换题目
- 点击答题卡题号快速跳转
- 点击"标记本题"标记疑问题目
- 系统每 30 秒自动保存进度

### 提交试卷
1. 答完所有题目后点击"提交试卷"
2. 如有未答题目会弹出提示
3. 确认提交后自动评分
4. 查看成绩并跳转到历史记录

### 继续未完成考试
- 如有关闭网页等中断情况
- 再次登录时主页会显示"继续未完成考试"提示
- 点击即可恢复到上次答题状态

### 查看历史记录
1. 点击主页"历史记录"或顶部导航
2. 查看所有已完成考试列表
3. 点击"查看详情"回看试卷和解析

## 自定义配置

### 修改默认密码
编辑 `config.php` 文件，修改以下常量：
```php
define('DEFAULT_PASSWORD', '123456'); // 改为你想要的密码
```
然后删除 `exam.db` 重新运行以重建用户表。

### 扩展题库
向 `questions` 表插入新题目，字段说明：
- `subject`: 科目（基础知识/理论与实务）
- `question_type`: 题型（single/multiple/case）
- `difficulty`: 难度（easy/medium/hard）
- `content`: 题目内容
- `options`: 选项（JSON 格式）
- `answer`: 正确答案
- `analysis`: 答案解析
- `score`: 分值
- `knowledge_point`: 知识点

## 注意事项

1. **数据安全**：定期备份 `exam.db` 文件以防数据丢失
2. **浏览器兼容**：推荐使用 Chrome、Firefox、Edge 等现代浏览器
3. **离线运行**：系统完全离线运行，无需网络连接
4. **单账号设计**：仅支持单账号使用，无多用户管理功能
